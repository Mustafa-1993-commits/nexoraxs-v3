# UI Review Template

Review goal:
Assess UI/UX quality for the requested NexoraXS screen or flow.

Check:
- clarity
- hierarchy
- spacing
- typography
- primary action
- empty/loading/error states
- responsive behavior
- accessibility basics
- consistency with platform/app role

For core-app:
- must feel like platform shell
- should not include app-specific business workflows

For admin-app:
- must feel like internal SaaS admin console
- should not include customer daily business workflows

For shops-app:
- must feel app-specific
- foundation phase should not show fake full modules

Output:
1. What works.
2. What feels confusing.
3. Recommended improvements.
4. Minimal implementation changes.
5. What to avoid.
