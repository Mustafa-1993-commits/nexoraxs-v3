# ADR-001: Establish Project Genesis

**Status:** Accepted
**Date:** 2026-07-05

---

## Context

Nexoraxs is evolving from a single SaaS app idea into a Business Operating Intelligence Platform.

The project needs a stable foundation so future specifications and code changes remain aligned.

---

## Decision

Create Project Genesis as the source of truth for:

- Vision
- Constitution
- Glossary
- Business DNA
- Capability Model
- Knowledge Engine
- Business Brain
- Recommendation Engine
- AI Strategy
- Roadmap

---

## Consequences

Future specifications should reference Genesis.

If product direction changes, Genesis should be updated before downstream specifications and implementation.
