1. Platform Overview
What is NexoraXS

NexoraXS is a modular SaaS ecosystem designed to host multiple independent business applications under one scalable cloud platform.

The platform provides:

Authentication system
Workspace management
App launcher
Subscription & billing system
Shared SaaS infrastructure
Independent business applications
Shared UI & shared packages
Multi-tenant architecture
2. Core Philosophy
NexoraXS is NOT a giant ERP monolith.

NexoraXS is built as:

Core Platform Shell
+
Independent Domain Applications
+
Shared Backend Infrastructure
+
Shared Packages

Goal:

Scalable SaaS ecosystem
Business operating system
Cloud-native platform
Future AI-ready architecture

17. Shared Authentication Across Subdomains
Important Configuration

Because applications use independent subdomains:

shops.nexoraxs.com
clinics.nexoraxs.com
app.nexoraxs.com

authentication must be shared across subdomains.

Required Session Configuration

Laravel:

SESSION_DOMAIN=.nexoraxs.com
Sanctum Configuration
SANCTUM_STATEFUL_DOMAINS=
app.nexoraxs.com,
shops.nexoraxs.com,
clinics.nexoraxs.com,
maintenance.nexoraxs.com,
admin.nexoraxs.com
CORS Configuration
ALLOWED_ORIGINS=
https://app.nexoraxs.com,
https://shops.nexoraxs.com,
https://clinics.nexoraxs.com,
https://maintenance.nexoraxs.com,
https://admin.nexoraxs.com
