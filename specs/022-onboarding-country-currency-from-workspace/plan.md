# Implementation Plan: Onboarding Country & Currency From Workspace

**Branch**: `023-onboarding-country-currency-from-workspace` | **Date**: 2026-05-14 | **Spec**: [spec.md](./spec.md)

## Summary

Three targeted edits across two apps: (1) Core Platform onboarding replaces Currency with Country (field + type + session write), adding a standalone `core_workspace_country` session key; (2) Shops Step 3 removes editable Country and Currency selects, adds read-only workspace context display with derived currency from a local static mapping; (3) Shops Step 4 Review adds workspace country and derived currency as inherited display values. `StoreSetupData` simplifies to `{ storeName, branch }`. No new packages, no new routes, no new files.

## Detailed Implementation Notes

### 1. `apps/core-platform/lib/session.ts`

```typescript
// Replace currency with country in WorkspaceSetup:
export interface WorkspaceSetup {
  workspaceName: string;
  slug: string;
  region: string;
  country: string;      // ← was: currency: string
  shopsEnabled: boolean;
}

// Add standalone write helper:
export function saveWorkspaceCountry(country: string): void {
  if (typeof window === "undefined") return;
  sessionStorage.setItem("core_workspace_country", country);
}
```

**Why a standalone key**: Shops reads `core_workspace_country` via `sessionStorage.getItem("core_workspace_country")` — a plain string, no JSON parsing. This avoids coupling Shops to the structure of the full `core_workspace_setup` JSON blob.

### 5. `apps/shops-app/app/onboarding/page.tsx`

**Add constant** (at module level):
```typescript
const COUNTRY_CURRENCY_MAP: Record<string, string> = {
  "Egypt":                 "EGP",
  "Saudi Arabia":          "SAR",
  "United Arab Emirates":  "AED",
  "Kuwait":                "KWD",
  "Qatar":                 "QAR",
};
const DEFAULT_WORKSPACE_COUNTRY = "Egypt";
const DEFAULT_WORKSPACE_CURRENCY = "EGP";

function getCurrencyForCountry(country: string): string {
  return COUNTRY_CURRENCY_MAP[country] ?? DEFAULT_WORKSPACE_CURRENCY;
}
```
