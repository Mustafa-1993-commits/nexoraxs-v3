# Implementation Plan: Onboarding Branch Country & Currency

**Branch**: `024-onboarding-branch-country-currency` | **Date**: 2026-05-14 | **Spec**: [spec.md](./spec.md)

## Summary

Replace the workspace-inherited country/currency pattern in Shops onboarding with explicit user-selected branch country and branch currency. `StoreSetupData` gains two new fields (`branchCountry`, `branchCurrency`). `StepStoreSetup` removes workspace props and adds two editable selects with a country→currency auto-suggest. `StepReview` removes workspace props and shows branch-scoped labels. The orchestrator removes all workspace country derivation. Dashboard components are unchanged — they already read the same session keys (`shops_country`, `shops_currency`) that this feature now populates from user input rather than workspace inheritance.

## Detailed Implementation Notes

### 1. `StepStoreSetup.tsx` — Major Update

**`StoreSetupData` interface** (extended):
```typescript
export interface StoreSetupData {
  storeName: string;
  branch: string;
  branchCountry: string;
  branchCurrency: string;
}
```

**Remove from props interface**:
- `workspaceCountry: string`
- `workspaceCurrency: string`

**Add constants**:
```typescript
const BRANCH_COUNTRY_CURRENCY_MAP: Record<string, string> = {
  "Egypt":                "EGP",
  "Saudi Arabia":         "SAR",
  "United Arab Emirates": "AED",
  "Kuwait":               "KWD",
  "Qatar":                "QAR",
};
const countryOptions = ["Egypt", "Saudi Arabia", "United Arab Emirates", "Kuwait", "Qatar"] as const;
const currencyOptions = ["EGP", "SAR", "AED", "KWD", "QAR", "USD", "EUR"] as const;
```

**Auto-suggest handlers** (inside component function, not JSX):
```typescript
const handleCountryChange = (newCountry: string) => {
  onChange({
    ...data,
    branchCountry: newCountry,
    branchCurrency: BRANCH_COUNTRY_CURRENCY_MAP[newCountry] ?? "EGP",
  });
};
```
Direct currency change: `onChange({ ...data, branchCurrency: event.target.value })`

**Workspace context row** — simplify to show name only (remove country/currency display):
```tsx
<div className="text-sm font-medium text-white">Workspace: Mustafa&apos;s Co.</div>
<div className="text-xs text-white/40">(Read-only)</div>
```

**Branch country select** — label "Branch country", `value={data.branchCountry}`, `onChange={handleCountryChange}`:
```tsx
<label className="space-y-2">
  <span className="text-sm font-medium text-white">Branch country</span>
  <select value={data.branchCountry} onChange={(e) => handleCountryChange(e.target.value)}
    className="w-full rounded-xl border border-white/10 bg-[#0a0a0f] px-4 py-3 text-sm text-white ...">
    {countryOptions.map(c => <option key={c} value={c}>{c}</option>)}
  </select>
</label>
```

**Branch currency select** — label "Branch currency", `value={data.branchCurrency}`, `onChange` updates only currency:
```tsx
<label className="space-y-2">
  <span className="text-sm font-medium text-white">Branch currency</span>
  <select value={data.branchCurrency} onChange={(e) => onChange({ ...data, branchCurrency: e.target.value })}
    className="...">
    {currencyOptions.map(c => <option key={c} value={c}>{c}</option>)}
  </select>
</label>
```

**Layout**: Branch country and currency selects in a `grid gap-5 sm:grid-cols-2` row below Store Display Name and Main Branch.

**Contextual note** (below the selects):
```tsx
<p className="rounded-xl border border-white/10 bg-white/[0.02] px-4 py-3 text-xs leading-relaxed text-white/50">
  Branch country and currency apply to your first branch. You can add more branches from{" "}
  <span className="text-white/70">Shops Settings → Branches</span>.
</p>
```

**Live preview card** — update rows to show branch values:
```tsx
<Badge color="blue">{data.branchCurrency}</Badge>
<span className="text-sm text-white">{data.branchCountry}</span>
```
Update row labels: "Branch currency" and "Branch country" (remove "Workspace country").
